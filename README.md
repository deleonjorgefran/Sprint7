# Sprint7
Tripleten

Este proyecto es una aplicación web interactiva desarrollada con Streamlit para el anánlisis de datos de anuncios de venta de vehículos en Estados Unidos. 

La app permite explorar el conjunto de datos de forma visual mediante: 

    - Histograma del odometro para observar su distribución
    - Gráfico de disperción entre odometro y precio para analizar la relación entre odometro y precio. 

El objetivo es facilitar una expñloración rápida y clara de los datos utilizando visualizaciones interactivas con "Plotly Express", apoyadas en el procesamiento de datos con "Pandas". 

Tecnologías utilizadas:
    - Python
    - Pandas
    - Streamlit
    - Plotly Express
    - nbformat
